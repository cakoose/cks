Data - Example CKS type and value definitions.

Data/Test - The test cases used by the automated test runner.

Implementation - CKS runtime libraries for different programming
  languages.

Implementation/Java - CKS runtime library for Java AND the CKS bindings
  generators for Java, C#, and Haskell.

Implementation/C# - CKS runtime library for C#

Implementation/Haskell - CKS runtime library for Haskell

Implementation/Test - Automated test runner for all languages (written
  in Java).

Website - Sources for CKS website.  Converted to HTML via the Ruby
  tool "WebGen".
